# ANN Classification Project

This project implements an Artificial Neural Network (ANN) for customer churn classification using TensorFlow and Scikit-learn.

## Tech Stack
- Python
- TensorFlow / Keras
- Scikit-learn
- Pandas
- NumPy

## Project Files
- app.py
- Churn_Modelling.csv
- model.h5
- scaler.pkl
- label_encoder_gender.pkl
- onehot_encoder_geo.pkl
- experiments.ipynb
- prediction.ipynb
- requirements.txt

## How to Run
```bash
pip install -r requirements.txt
python app.py
